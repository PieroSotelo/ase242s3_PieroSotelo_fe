**¿Qué es Angular?**
Angular es un framework(conjunto de herramientas predefinidos incluyendo librerias y diferentes cosas) de código abierto para poder realizar desarrollos de diferentes aplicaciones web, de esta manera incluye diferentes lenguajes para los estilos para de esta manera poder facilitar las futuras creaciones de aplicaciones web, siendo de este modo una gran ayuda. 
**¿Qué herramientas necesitan instalar?** 
Para poder realizar la instalación de Angular debemos tener 2 cosas importantes primero descargar el node.js y npm, pero para poder tener esto no es necesario instalarlo por separado, con solo descargar el node.js viene incluido tambien el npm lo cual solo deberiamos instalar este para poder trabajar con angular y luego para poder ejecutar código y editar usaremos vsc.
